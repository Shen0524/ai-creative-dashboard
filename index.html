「問候大家營養沒」系列

已完成 0 / 16
方向

視覺設定

  • Q 版 LINE 貼圖
  • 可愛／搞笑／浮誇表情擇一
  • 以營養與日常問候為主題
  • 口吻可參考「呷飽沒」
可直接使用

核心提示詞

製作一套可愛 Q 版 LINE 貼圖，主題為「問候大家營養沒」。設計 16 組貼近日常、簡短好讀的營養問候題詞，例如「呷飽沒」，搭配可愛、搞笑或浮誇的表情與動作。每張構圖清楚、角色一致、文字醒目，適合聊天情境使用。
